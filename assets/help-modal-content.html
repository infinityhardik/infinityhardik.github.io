1. Start Typing :
    Begin typing the Product Name to Filter the list dynamically. The search is not case-sensitive, meaning that "MRX", "Mrx", "mrX" or "mrx" will all search for "MRX".

    Examples :
    1. To search for MRX F/D 30mm 90 x 36, type 9036. Other options: 309036, M309036, MX309036, MRX309036
    2. To search for ZVK XL 18 mm 8 x 4, type 1884. Other options: z1884, zvk1884, xl1884
    3. To filter for GT GJ Products, type GTGJ. Other options: GT, GJ, GTG, GGJ
  2. Order Quantity :
    Input the Quantity as required to generate your order.
    • • Press Enter to add the quantity for the selected item.
    • • Press Delete to clear the quantity (set to zero).
    PRO Tip for Instant Input :

    Use the Search Bar directly : Type the search term followed by a hyphen "-" and the quantity.

    1. GTGJ1884-10 : Adds the first visible item with 10 Qty.
    2. M359648-5 : Adds MRX F/D 35 mm 96 x 48 with 5 Qty.
  3. Ctrl + Enter :
    Send Order through WhatsApp.
  4. Ctrl + Backspace :
    Clear the search box quickly (or use the X button).
  5. Ctrl + Delete :
    Clear the Entire Order.
  6. Ctrl + Shift + F :
    Open Filter Options for Product List.
  7. Ctrl + Shift + O :
    Clear Filters and show all products.
  8. Ctrl + O :
    Toggle View/Edit Order popup.
  9. Ctrl + S :
    Toggle Show Selected Only in Product List.
  10. Ctrl + / or Ctrl + ? :
    Open Help Text and Shortcuts.
  11. Ctrl + ↑ / → :
    Increase quantity of the focused item.
  12. Ctrl + ↓ / ← :
    Decrease quantity of the focused item.
  13. Ctrl + H :
    Open Order History.
  14. Order History Feature :
    Orders are saved for 7 days. You can:
    • • View : See full details of past orders.
    • • Edit : Load previous orders to modify and resend.
    • • Copy : Copy order text directly.
    • • Search : Find orders by product, date, or quantity.
Tip : Use Filter Options to search specific categories like Plywood, Block-Board or Flush-Doors.

  1. टाइप करना शुरू करें :
    प्रोडक्ट लिस्ट को फ़िल्टर करने के लिए नाम टाइप करना शुरू करें। खोज केस-सेंसिटिव नहीं है, यानी "MRX" या "mrx" सभी "MRX" के लिए खोजेंगे।

    उदाहरण :
    1. MRX F/D 30mm 90 x 36 के लिए 9036 टाइप करें। अन्य विकल्प: 309036, M309036
    2. ZVK XL 18 mm 8 x 4 के लिए 1884 टाइप करें। अन्य: 1884, Z1884
    3. GT GJ प्रोडक्ट्स के लिए GTGJ टाइप करें।
  2. ऑर्डर मात्रा :
    आवश्यक मात्रा दर्ज करें और ऑर्डर जनरेट करें।
    • • चयनित आइटम के लिए मात्रा जोड़ने हेतु Enter दबाएं।
    • • मात्रा शून्य करने हेतु Delete दबाएं।
    त्वरित इनपुट के लिए टिप :

    सर्च बार का सीधा उपयोग करें: नाम के बाद हाइफ़न "-" और मात्रा लिखें।

    1. GTGJ1884-10 : पहले दिखाई देने वाले आइटम को १० मात्रा के साथ जोड़ देगा।
    2. M359648-5 : MRX F/D 35 mm 96 x 48 को ५ मात्रा के साथ जोड़ देगा।
  3. Ctrl + Enter :
    WhatsApp के माध्यम से ऑर्डर भेजें।
  4. Ctrl + Backspace :
    खोज बॉक्स को जल्दी खाली करें।
  5. Ctrl + Delete :
    पूरा ऑर्डर खाली करें।
  6. Ctrl + Shift + F :
    लिस्ट के लिए फ़िल्टर विकल्प खोलें।
  7. Ctrl + Shift + O :
    फ़िल्टर साफ़ करें और सभी प्रोडक्ट्स दिखाएं।
  8. Ctrl + O :
    ऑर्डर देखें या संशोधित करें।
  9. Ctrl + S :
    लिस्ट में केवल चयनित दिखाएं।
  10. Ctrl + / या Ctrl + ? :
    मदद और शॉर्टकट्स खोलें।
  11. Ctrl + ↑ / → :
    चयनित आइटम की मात्रा बढ़ाएँ।
  12. Ctrl + ↓ / ← :
    चयनित आइटम की मात्रा कम करें।
  13. Ctrl + H :
    ऑर्डर हिस्ट्री खोलें।
  14. ऑर्डर हिस्ट्री :
    ऑर्डर ७ दिनों के लिए सहेजे जाते हैं। आप कर सकते हैं:
    • • देखें : पिछले ऑर्डर का पूरा विवरण देखें।
    • • संपादित करें : पुराने ऑर्डर को लोड कर संशोधित करें।
    • • कॉपी : सीधे ऑर्डर टेक्स्ट कॉपी करें।
    • • खोज : उत्पाद या तारीख से ऑर्डर खोजें।
टिप : विशिष्ट श्रेणियों (प्लाईवुड, बोर्ड) के लिए फ़िल्टर का उपयोग करें।
