    How to Search for Products and Shortcuts
    1. Start Typing :
      Begin typing the Product Name to Filter the list dynamically. The search is not case-sensitive meaning that "MRX", "Mrx", "mrX" or "mrx" will search for "MRX".
      Examples will help you understand better :

      1 : To Search for MRX F/D 30mm 90 x 36, type 9036 in search box to get to this product faster. Other Search Options can be : 309036, M309036, MX309036, MRX309036, MRX30mm9036

      2 : To Search for ZVK XL 18 mm 8 x 4, type 1884 in search box to get to this product faster. Other Search Options can be : 1884, Z1884, ZV1884, ZVK1884, ZVKXL1884, XL1884, ZXL184

      3 : To Filter List for GT GJ Products, type GTGJ in search box to Filter the Product List with GT GJ Products. Other Search Options can be : GT, GJ, GTGJ, GTG, GTJ, GGJ

    2. Order Quantity :
      Input the Quantity as required and your order will get generated.

      Press Enter to Add the Order Quantity for the selected Product Item to the List.

      Press Delete to Clear the Order Quantity for the selected Product Item to the List. The Selected Item Quantity becomes Zero.

    3. Ctrl + Enter :
      Send Order through WhatsApp.
    4. Ctrl + Backspace :
      Clear the search box quickly using Shortcut or X Button.
    5. Ctrl + Delete :
      Clear the Entire Order.
    6. Ctrl + Shift + F :
      Open Filter Options for Product List.
    7. Ctrl + Shift + O :
      Clear Filter Options for Product List and show all the Products.

    8. Ctrl + / :

      OR

      Ctrl + ? :

      Open Help Text and Shortcuts.

    Tip : Use the Filter Options to List specific type of results for search.
    Example : Filter Options can be used to Filter List based on Product Type of Plywood, Block-Board or Flush-Door for search.

    1. टाइप करना शुरू करें :
      प्रोडक्ट लिस्ट को फ़िल्टर करने के लिए प्रोडक्ट का नाम टाइप करना शुरू करें। खोज केस-सेंसिटिव नहीं है, यानी "MRX", "Mrx", "mrX" या "mrx" सभी "MRX" के लिए खोजेंगे।
      उदाहरण आपको बेहतर समझने में मदद करेंगे:

      १ : MRX F/D 30mm 90 x 36 के लिए खोजने हेतु, 9036 टाइप करें। अन्य खोज विकल्प: 309036, M309036, MX309036, MRX309036, MRX30mm9036

      २ : ZVK XL 18 mm 8 x 4 के लिए खोजने हेतु, 1884 टाइप करें। अन्य खोज विकल्प: 1884, Z1884, ZV1884, ZVK1884, ZVKXL1884, XL1884, ZXL184

      ३ : GT GJ प्रोडक्ट्स के लिए लिस्ट फ़िल्टर करने हेतु, GTGJ टाइप करें। अन्य खोज विकल्प: GT, GJ, GTGJ, GTG, GTJ, GGJ

    2. ऑर्डर मात्रा :
      आवश्यक मात्रा दर्ज करें और आपका ऑर्डर जनरेट हो जाएगा।

      चयनित प्रोडक्ट के लिए ऑर्डर मात्रा जोड़ने के लिए Enter दबाएं।

      चयनित प्रोडक्ट की मात्रा को शून्य करने के लिए Delete दबाएं।

    3. Ctrl + Enter :
      WhatsApp के माध्यम से ऑर्डर भेजें।
    4. Ctrl + Backspace :
      शॉर्टकट या X बटन से खोज बॉक्स को जल्दी खाली करें।
    5. Ctrl + Delete :
      पूरा ऑर्डर खाली करें।
    6. Ctrl + Shift + F :
      प्रोडक्ट लिस्ट के लिए फ़िल्टर विकल्प खोलें।
    7. Ctrl + Shift + O :
